## ğŸ“ File Naming

### Documentation

- Use UPPERCASE for root docs: `README.md`, `CONTRIBUTING.md`, `SECURITY.md`
- Use lowercase with hyphens for docs/: `quick-start.md`, `data-flow.md`
- Use UPPERCASE for special docs: `BRANDING.md`, `ROADMAP.md`, `CONVERGENCE.md`

### Code

- Use lowercase with hyphens for source files: `wave-analysis.ts`, `api-client.ts`
- Use camelCase for exported functions: `analyzeWave()`, `validateBump()`
- Use SCREAMING_SNAKE for constants: `PHI`, `LAMBDA_THRESHOLD`
