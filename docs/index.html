Coherence MCP

Model Context Protocol server for AI coherence verification. Wave analysis, ATOM tracking, and human-AI collaboration primitives.

Get Started npm Package

npx @hopeandsauced/coherence-mcp

Installation

Claude Desktop

Add to claude_desktop_config.json:

{
  "mcpServers": {
    "coherence": {
      "command": "npx",
      "args": ["@hopeandsauced/coherence-mcp"]
    }
  }
}

Global Install

npm install -g @hopeandsauced/coherence-mcp

Then run:

coherence-mcp
